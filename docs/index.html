Install

Option 1 — TestFlight (easiest, when available)

If a TestFlight build is live, the trip leader will send a separate invite link. Tap it on your iPhone, install TestFlight from the App Store if prompted, then accept the GE136C invite.

Option 2 — Build it yourself with Xcode

Free, no developer account needed beyond your Apple ID. On a Mac:

1. Clone the repo: git clone https://github.com/yicolas/festy.git
2. Open bitchat.xcodeproj in Xcode.
3. Plug in your iPhone, pick it as the destination, hit ⌘R.
4. On the iPhone: Settings → General → VPN & Device Management → trust the developer certificate.

Option 3 — Sideloading tools

Advanced users can use AltStore / SideStore / Sideloadly to install a signed .ipa from the GitHub Releases page. Requires a Mac or AltServer running.

What it does

Offline mesh chat over Bluetooth — no cell service needed.
Trip channels (#general, #cars, #meals, etc.) keep coordination organized.
Pre-downloadable OSM topo tiles + multi-day driving routes for offline navigation.
End-to-end encrypted trip chat — strangers can't read your group messages.

Privacy

No accounts, no phone numbers, no tracking. Chat history, map tiles, and route caches are stored on your device and stay until you clear them. Nothing leaves the device except the encrypted mesh chat packets to other GE136C participants in Bluetooth range.
